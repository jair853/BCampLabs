Tabla de conceptos básicos para diseño web

World Wide Web (www)
Es la red mundial para transmitir información conocida como internet
Internet Protocol (IP)
Es un número o código que identifica cada equipo conectado a la internet
Domain Name System (DNS)
Es el sistema de nombres de dominio, diseñado para que las personas encuentren fácilmente los sitios web
