Estilos con CSS

Este es el laboratorio para trabajar agregando estilos a una página html usando tres opciones:

Estilos en línea:

Se hace dentro de la misma etiqueta, usando el atributo "style".

Estilos dentro de la sección <head>:

Se establece el estilo en la sección <head>, usando el bloque <style>.

Estilos en archivo CSS:

Se utiliza un archivo .css que permite editar y reutilizar la configuración de la página, facilitando el mantenimiento del código.

    Uso de los estilos en hojas de estilo.

  • Llamada a elementos específicos.
  • Llamada a conjuntos de elementos.
  • Reutilización de código.
